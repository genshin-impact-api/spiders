==============
Character Info
==============

This module allows people to retrieve playable character info, including

* Chinese/English Names
* Chinese/English Titles
* `Character Vision <https://genshin-impact.fandom.com/wiki/Vision>`_ (CN)
* the URLs of each character page in
  `原神WIKI <https://wiki.biligame.com/ys/%E9%A6%96%E9%A1%B5>`_ (CN)/`Fandom <https://genshin-impact.fandom.com/wiki/Genshin_Impact_Wiki>`_ (EN),
